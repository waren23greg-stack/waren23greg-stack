### 🏭 [Supply Chain Intelligence Platform](https://github.com/waren23greg-stack/supply-chain-intelligence)
> End-to-end supply chain analytics system built with **MySQL · Python · Power BI**

A full-stack data analytics pipeline simulating real-world BI workflows in logistics, procurement, and operations.

| Layer | Details |
|---|---|
| **Database** | 5 relational tables · 9,450 rows · 18 advanced SQL queries |
| **Python Engine** | EDA · Linear Regression forecasting · 7-panel chart dashboard |
| **Power BI** | 5 interactive pages · Executive KPIs · Supplier Scorecard · Inventory Health |
| **Collab Tool** | Node.js + Express browser UI with real-time SSE progress streaming |

**Key insights surfaced:**
- 💰 **$68.6M** in total revenue analyzed across 24 months
- 🔁 **25%** cancellation rate flagged for investigation
- 📦 **20 products** identified for immediate reorder ($194K reorder cost)
- 📈 Revenue forecast: **~$2.2M/month** for Q2 2026
